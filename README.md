# <img src="src/zlrc.png" width = "64" height = "40" alt="" align="left" /> zlrc

[English](README.en.md)

zlrc是一款基于[ZonyLrcTools](https://github.com/jjzhang166/ZonyLrcTools)的歌词下载工具，参考了该项目的部分核心源码。

仅支持Windows系统（≥win7）。
